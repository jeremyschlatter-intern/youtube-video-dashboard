About This Dashboard

The Problem

When House committees livestream or upload hearings, markups, and other proceedings to YouTube, the Library of Congress needs the official Event ID from docs.house.gov to link those videos on congress.gov. Without the Event ID in the video description, proceedings effectively go missing from the official legislative record.

At the June 2025 Congressional Data Task Force Meeting, the Library of Congress worked with the House Digital Service to identify unlabeled videos with their proper Event IDs. However, that effort only covered 2025 data. This dashboard extends that work to prior years and provides ongoing tracking.

How It Works

  • YouTube Scanning: We scan each House committee's YouTube channel for videos that appear to be full proceedings (45+ minutes, excluding press conferences and interviews).
  • Event ID Detection: We check each video's description for an Event ID (e.g., "EventID=118951").
  • Cross-Referencing: We pull official hearing data from the congress.gov API and match videos to official events using dates, titles, and committee membership.
  • Grading: Committees are graded on what percentage of their YouTube proceedings include Event IDs.

Grading Scale

The grading scale is calibrated to the current reality where most committees do not include Event IDs:

A50%+ of proceedings include Event IDs
B30-49% include Event IDs
C15-29% include Event IDs
D5-14% include Event IDs
FLess than 5% include Event IDs

Event ID Formats Used by Committees

Committees that include Event IDs use various formats in their video descriptions:

  • EventID=118951 (Appropriations style)
  • Event ID: 118951 (Judiciary Democrats style)
  • Embedded in docs.house.gov or congress.gov URLs

Background

This project was proposed as part of the Congressional Data Task Force initiative to improve access to legislative information. The goal is to encourage all committees to consistently include Event IDs, and to provide the Library of Congress with backfill data for prior years.

Data Sources

  • YouTube committee channels (metadata via yt-dlp)
  • Congress.gov API for official committee meeting data
  • docs.house.gov for event records and Event IDs
